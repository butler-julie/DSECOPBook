Lecture IV:#

\begin{equation} da := \frac{dL}{da} = - \frac{y^{i}}{a} + \frac{1- y^{i}}{1 - a} \end{equation}

\begin{equation} dz := \frac{dL}{dz} = \frac{dL}{da} * \frac{da}{dz} = [- \frac{y^{i}}{a} + \frac{1- y^{i}}{1 - 1}][a (1 -a )]= a - y^{i} \end{equation}

\begin{equation} d\omega_1 := \frac{dL}{d\omega_1} = \frac{dL}{da} * \frac{da}{dz}* \frac{dz}{d\omega_1} = (a - y^{i}) x_1^{i} \end{equation}

\begin{equation} d\omega_2 := \frac{dL}{d\omega_2} = \frac{dL}{da} * \frac{da}{dz}* \frac{dz}{d\omega_2} = (a - y^{i}) x_2^{i} \end{equation}

\begin{equation} db := \frac{dL}{db} = \frac{dL}{da} * \frac{da}{dz}* \frac{dz}{db} = (a - y^{i}) \end{equation}

Another way to regularize the cost function is to use both ‘\(L_1\)-Regularization’ and ‘\(L_2\)-Regularization’ simultaneously. This method is called Elastic net. The extra term in cost function in this method is

Lecture IX:#

An example of a linear activation function is \(A(Z) = Z\), which effectively means there is no activation function! So, in this case, we have

\begin{equation} A^{[l = 1]} = Z^{[l = 1]} = \omega^{[l = 1]}X_0 + b^{[l = 1]} \end{equation} \begin{equation} A^{[l = 2]} = Z^{[l = 2]} = \omega^{[l = 2]}[\omega^{[l = 1]}X_0 + b^{[l = 1]}] + b^{[l = 2]} \end{equation}

and so on. Therefore, for the last layer, we can write,

\begin{equation} A^{[L]} = Z^{[L]} = \omega^{[L]}\omega^{[L-1]}\omega^{[L-2]}…\omega^{[1]}X_0 + F(b^{[l]}, \omega^{[l]}) \end{equation}

Technically, we can define a new \(\omega\) and a new \(b\) as

\begin{equation} \omega_{new} = \omega^{[L]}\omega^{[L-1]}\omega^{[L-2]}…\omega^{[1]} \end{equation} \begin{equation} b_{new} = F(b^{[l]}, \omega^{[l]}) \end{equation}

Using these new parameters, we can rewrite the output as

\begin{equation} A^{[L]} = Z^{[L]} = \omega_{new}X_0 + b_{new} \end{equation}

Hence, effectively it is like a neural network with just one hidden layer.
